projects

  • DinnerSafe

    • React
    • API
    • Sass
    • Firebase
    • Authentication

    DinnerSafe finds recipes for a group of guests which match everyone’s dietary restrictions, which can be further filtered with search or meal-type. Built with React, the Yummly API, and Firebase, DinnerSafe can be used as a guest or with an account authenticated by GMail login. Users can save and edit events, guests and save recipes to their events.

  • Movie Mapper

    • jQuery
    • API
    • Sass
    • Namespaced

    MovieMapper displays the production countries and movie details for a movie selected by search or the “Random Movie” button. MovieMapper is built with JS namespacing, jQuery, and the Movie Database API. Inconsistent data from the API is handled with error messages and by hiding sections if only partial data is returned.

  • Hello, My Name Is

    • jQuery
    • Responive
    • Sass
    • Namespaced

    This game allows users to select a group of people and then quizzes them on names of those people, showing a picture with four names. Built with jQuery, and JS namespacing, this game will randomly select a new face from the question set until the timer runs out. Faces will not repeat until the entire set has been shown.

    View Live View on GitHub

  • Responsive PSD Conversion

    • Sass
    • BEM Methodology
